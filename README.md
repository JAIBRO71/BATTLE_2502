# Multi-Context Conflict Resolver

This project resolves conflicts between multiple user inputs, preferences, and constraints using Featherless AI.

## Features
- Detects conflicting requirements
- Prioritizes constraints
- Uses Featherless AI
- Gives final decision with explanation

## Tech Stack
- Python
- Featherless AI
- OpenAI SDK

## Run
pip install -r requirements.txt
python main.py
